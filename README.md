# Umer Azmi — Developer Portfolio

A modern, fully responsive personal developer portfolio built using HTML, CSS, and JavaScript.

This project showcases services, skills, experience, education, and dynamically rendered projects using structured JSON data. Designed with a clean dark theme, smooth animations, and performance-focused UI structure.

## ✨ Features

### Core UI

* Fully responsive layout (mobile, tablet, desktop)
* Modern dark theme with custom CSS variables
* Smooth scrolling behavior
* Fixed glass-style navigation with scroll effect
* Mobile slide-in navigation menu

### Home Section

* Animated typing text effect (pure CSS)
* Profile image with animated gradient ring
* Social media icons with hover glow effects
* Call-to-action buttons (Hire Me & View Work)

### Services

* Grid-based service cards
* Hover animations with glow and elevation effects
* Clean icon + numbering layout

### Skills

* Dynamically rendered from `data.json`
* Categorized technology groups
* Interactive skill chips with hover transitions

### Education & Experience

* Vertical timeline layout
* Scroll reveal animations using Intersection Observer
* Highlight tags for technologies and tools

### Projects

* Dynamic rendering from JSON
* Filter system (All / Live / In Development / Concept)
* Status badges
* Video preview on hover
* Responsive grid layout

### Contact

* Contact information section
* Styled contact form UI
* Static form submission message
* Animated contact cards

## 🚀 How to Run Locally

1. Clone or download the repository.
2. Ensure `data.json` is in the same directory as `index.html`.
3. Open `index.html` in a modern browser.

Recommended (VS Code users):

* Install Live Server extension
* Right-click `index.html`
* Select "Open with Live Server"
